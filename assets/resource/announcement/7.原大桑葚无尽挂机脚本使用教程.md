# 【原大桑葚无尽挂机脚本】使用教程

感谢您使用 **原大桑葚无尽挂机脚本**！本脚本专为无尽模式挂机设计，自动完成抛花（普通关）、Boss关喂豆,补种植物等关键操作，解放双手。

## 一、脚本核心功能

| 功能             | 说明                                                                 |
|------------------|----------------------------------------------------------------------|
| 普通关抛花（可选） | 每小关自动抛 **2 朵** 能量花，抛花位置固定：**9-1**、**9-2**（右上角为9-1） |
| Boss关喂豆（可选） | Boss关 **一直喂豆直到结算**，喂豆位置固定：**(5-3)**（可放输出/控制类植物） |
| 选卡逻辑     | 开始前重新选卡，优先选择 **桑葚、毒藤、大哥**，除卡槽上必要植物其余全选橙色植物   |
| 卡槽容错         | 若没有“大哥”，则将大哥卡槽替换为 **泡泡姬珊瑚**                       |

## 二、使用前准备（重要！）

### 1. 卡槽配置![阵容图](image/yd_卡槽.png)

- 脚本依赖 **固定卡槽顺序**，请确保以下植物位于对应卡槽(后两个植物随意)：

### 2. 抛花前提

- 普通关每小关抛花 **2 朵**，抛花位置固定：
  - 格子 `(9-1)` → 右上角第一格
  - 格子 `(9-2)` → 右上角第二格
- 请确保这两个格子 **空余或可丢弃能量花**，否则抛花可能失败。

### 3. Boss关喂豆前提

- 喂豆位置固定：`(5-3)`
- 5-3种的植物为你卡槽的第7个,可以自己换植物(建议换控制和输出植物)
- 建议在该格子放置 **输出类** 或 **控制类** 植物（如地刺,磁铁等），脚本会一直喂豆直到Boss关结算。

### 4. 选卡规则（脚本自动执行）

| 优先级 | 植物选择                           | 备注                         |
|--------|------------------------------------|------------------------------|
| 1      | 桑葚、毒藤、大哥                   | 优先选这三个，缺一不可         |
| 2      | 其他（除必备植物外）全选橙色植物 | 尽量选橙色植物品质               |
| 容错   | 若无大哥 → 将大哥卡槽换成泡泡姬珊瑚 | 脚本自动处理，无需手动干预     |
